## Global

#### `authorityUrl*`
> **type**: string<br/>
> **required**

#### `clientId*`
> **type**: string<br/>
> **required**

#### `loginRequired`
> **type**: boolean<br/>
> **default**: true

#### `retrieveUserSession`
> **type**: boolean<br/>
> **default**: true

#### `loadUserInfo`
> **type**: boolean<br/>
> **default**: false

#### `automaticSilentRenew`
> **type**: boolean<br/>
> **default**: true

#### `desktopNavigationType`
> **type**: enum - *DesktopNavigation.(REDIRECT, POPUP)*<br/>
> **default**: REDIRECT

#### `scope`
> **type**: string<br/>
> **default**: "openid profile email phone"

#### `logLevel`
> **type**: enum - *Log.(NONE, ERROR, WARN, INFO, DEBUG)*<br/>
> **default**: NONE

#### `internal`
> **type**: UserManagerSettings<br/>


## Angular

#### `automaticInjectToken`
> **type**: boolean or custom object<br/>
> **default**: any hostname or pathname that starts with 'api'

```ts title="Custom object example"
automaticInjectToken: {
  include: (url: string): boolean => url?.includes('user'),
  exclude: [
    '/public-api',
    /^\/my-custom-api\/.*$/gm
  ]
}
```

#### `automaticLoginOn401`
> **type**: boolean<br/>
> **default**: true

#### `authGuardRedirectUrl`
> **type**: string<br/>
