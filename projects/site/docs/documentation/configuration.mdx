## Global

#### `authorityUrl*`
> **type**: string<br/>
> **required**

#### `clientId*`
> **type**: string<br/>
> **required**

#### `loginRequired`
> **type**: boolean<br/>
> **default**: true

#### `retrieveUserSession`
> **type**: boolean<br/>
> **default**: true

#### `loadUserInfo`
> **type**: boolean<br/>
> **default**: false

#### `automaticSilentRenew`
> **type**: boolean<br/>
> **default**: true

#### `navigationType`
> **type**: enum - *Navigation.(REDIRECT, POPUP)*<br/>
> **default**: REDIRECT

#### `scope`
> **type**: string<br/>
> **default**: "openid profile email phone"

#### `logLevel`
> **type**: enum - *Log.(NONE, ERROR, WARN, INFO, DEBUG)*<br/>
> **default**: NONE

#### `internal`
> **type**: UserManagerSettings<br/>


## Angular

#### `automaticInjectToken`
> **type**: boolean or custom object<br/>
> **default**: see the examples

```ts title="Examples"
// default: any url that starts with "/api"
automaticInjectToken: {
  include: (url: string): boolean => {
    const matches = new RegExp(/^.*?(?<!\/)\/(?!\/)(.*$)/gm).exec(url);
    return (matches?.[1]?.startsWith('api')) || false;
  }
}
```
